# Bar Raiser Winner '26 — Profile Frame App

A single-page web app that lets Delhivery Bar Raiser Winners apply a custom frame to their profile photo and download it as a circular transparent PNG.

## Deployment (GitHub Pages)
1. Upload all files to the repository root
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. Save — your site will be live at `https://yourusername.github.io/your-repo-name`

## Files
- `index.html` — the entire self-contained app (all assets embedded)
- `README.md` — this file
