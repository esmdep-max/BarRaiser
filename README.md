# Bar Raiser Winner — Profile Frame App

A single-page web app that lets Delhivery Bar Raiser Winners apply a custom frame to their profile photo and download it as a circular transparent PNG.

## Deployment

### GitHub Pages (recommended)
1. Create a new GitHub repository
2. Upload all files from this folder to the repository root
3. Go to **Settings → Pages**
4. Set source to **Deploy from a branch → main → / (root)**
5. Save — your site will be live at `https://yourusername.github.io/your-repo-name`

### Netlify
1. Go to [netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop this entire folder
3. Your site is live instantly

## How it works
- User uploads their profile photo
- The Bar Raiser Winner frame (embedded in the HTML) is auto-applied
- User downloads a circular PNG ready to use as a profile picture
